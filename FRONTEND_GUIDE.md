# 前端界面使用指南

## 🚀 快速开始

### 方式 1: 直接在浏览器中打开

1. 确保服务器正在运行：
   ```bash
   python main.py
   ```

2. 直接在浏览器中打开 `index.html` 文件：
   ```
   双击 index.html 文件
   ```

3. 在浏览器地址栏中访问：
   ```
   file:///E:/path/to/claude-code/index.html
   ```

### 方式 2: 使用本地服务器（推荐）

**使用 Python：**
```bash
# 使用 Python 简单 HTTP 服务器
python -m http.server 8080

# 在浏览器访问
http://localhost:8080
```

**使用 VS Code Live Server：**
1. 安装 Live Server 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 📱 功能说明

### 📄 文件上传功能

**支持的操作：**
- ✅ 点击上传区域选择文件
- ✅ 拖拽文件到上传区域
- ✅ 支持的格式：.txt, .md, .pdf
- ✅ 显示文件信息（名称、大小、类型）
- ✅ 上传进度和成功提示

**使用流程：**
1. 切换到"文件上传"标签页
2. 选择要上传的文档
3. 点击"开始处理"按钮
4. 等待上传完成
5. 上传成功后会显示文本块数量

### 💬 智能问答功能

**使用流程：**
1. 切换到"智能问答"标签页
2. 在输入框中输入问题
3. 点击发送按钮或按 Enter 键
4. 系统会：
   - 从数据库中检索相关内容
   - 调用 LLM 生成答案
   - 显示问答记录

**支持的对话：**
- 定义类问题
- 应用场景问题
- 工具介绍问题
- 模型列举问题

## 🎨 界面特性

### 设计特点

- **现代渐变背景**: 紫色渐变设计
- **响应式布局**: 适配桌面和移动设备
- **动画效果**: 流畅的消息出现动画
- **状态指示器**: 实时显示服务状态
- **加载动画**: 上传和提问时的加载状态

### 颜色方案

- **主色调**: 紫色渐变 (#667eea → #764ba2)
- **用户消息**: 紫色背景
- **AI 回复**: 灰色背景
- **系统消息**: 浅灰色背景
- **状态点**:
  - 🟢 绿色 = 服务运行中
  - 🟡 黄色 = 加载中
  - 🔴 红色 = 服务未连接

## 🌐 技术栈

- **HTML5**: 语义化标签
- **CSS3**: Flexbox 布局、渐变、动画
- **Vanilla JavaScript**: 无依赖，使用 fetch API
- **API**: FastAPI 后端接口

## 🔧 API 配置

前端默认配置：
```javascript
const API_BASE = 'http://localhost:8000';
```

如需修改 API 地址，在 `index.html` 中找到第 529 行并修改：
```javascript
const API_BASE = 'http://your-api-url:port';
```

## 🔍 故障排除

### 问题 1: 服务器连接失败

**症状**: 状态点显示红色，无法发送消息

**解决方案**:
1. 确保服务器正在运行：
   ```bash
   python main.py
   ```

2. 检查 API 地址是否正确（默认 `http://localhost:8000`）

3. 查看浏览器控制台错误信息

### 问题 2: 文件上传失败

**症状**: 上传按钮显示错误

**解决方案**:
1. 检查文件格式是否为 .txt, .md, .pdf
2. 确保文件大小不超过服务器限制
3. 查看服务器日志中的错误信息

### 问题 3: 问答无响应

**症状**: 点击发送后没有反应

**解决方案**:
1. 确保已上传文档
2. 检查输入框是否有内容
3. 查看浏览器控制台错误

## 🎯 使用技巧

1. **先上传文档**: 在"文件上传"标签页上传文档后再提问
2. **多轮对话**: 可以持续提问，系统会记住上下文
3. **精简问题**: 问题越具体，答案越准确
4. **查看文档**: 定期切换到文件上传标签页查看已上传的文档
