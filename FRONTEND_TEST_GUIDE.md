## 🚀 测试方法

### 方法 1: 手动测试（推荐）

1. **启动服务器**
   ```bash
   python main.py
   ```

2. **打开前端页面**
   - 方式 A: 双击 `index.html` 文件
   - 方式 B: 使用本地服务器
     ```bash
     python -m http.server 8080
     # 访问: http://localhost:8080
     ```

3. **测试步骤**
   - 切换到"文件上传"标签页
   - 拖拽或选择一个测试文件
   - 点击"开始处理"
   - 等待上传成功
   - 切换到"智能问答"标签页
   - 输入问题并提问

### 方法 2: API 测试

```python
import requests

# 上传文件
files = {'file': ('test.txt', '测试内容')}
response = requests.post('http://localhost:8000/upload', files=files)
print(response.json())

# 问答
payload = {'question': '测试问题'}
response = requests.post('http://localhost:8000/ask', json=payload)
print(response.json())
```

## 🔍 问题诊断

### 问题 1: 页面无法加载

**可能原因:**
- 文件路径错误
- 文件编码问题

**解决方法:**
```bash
# 检查文件
file index.html

# 使用正确的编码打开
# Windows: 右键 -> 打开方式 -> Chrome
```

### 问题 2: 上传失败

**检查:**
1. 服务器是否运行
2. 文件格式是否正确（.txt, .md, .pdf）
3. 文件大小是否太大

### 问题 3: 问答无响应

**检查:**
1. 文档是否已上传
2. 输入框是否可用
3. 服务器日志中的错误信息
